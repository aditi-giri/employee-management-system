# Employee Management System

A full-stack **Employee Management System** built using **Spring Boot, React, and MySQL**.
The application allows users to manage employee records through a simple web interface.

## Tech Stack

* **Backend:** Spring Boot, Spring Data JPA
* **Frontend:** React
* **Database:** MySQL

## Features

* Add new employees
* View employee list
* Update employee details
* Delete employees
* REST API integration between frontend and backend

## Project Structure

```
backend   -> Spring Boot REST API
frontend  -> React application
```

## How to Run

### Backend

```
cd backend
./mvnw spring-boot:run
```

### Frontend

```
cd frontend
npm install
npm run dev
```
